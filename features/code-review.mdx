title: 'Code Review'
description: 'How to get a full line by line code review from Squire'
icon: 'mug-hot'
---

When on an open PR you can invoke Squire with `@squire /review` then click add single comment and we will conduct a line by line review.

*Please Note*: you can only invoke this on .ts & .tsx files today, though we will review the entire PR
