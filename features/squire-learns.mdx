Once Squire has completed [a code review](/features/code-review), you can help teach Squire to provide even better reviews for you.

You teach Squire by reacting to any Squire comments:
- a thumbs up emoji 👍 for "more comments like these" or;
- a thumbs down emoji 👎 for "less comments like these"
